Um site de bate-papo como aprendizado de NodeJS.
Para rodar você precisa ter instalado NodeJS.

NPM's instalados:

Socket.io
Express
Express-validator
Consign
bodyParser
Nodemon

Para colocar ao ar no localhost, basta ir para CMD e redirecionar para pasta a onde
está localizado os arquivos, depois escrever "nodemon app" e apertar o ENTER.

Para visualizar bastar escrever localhost como o http no navegador.
